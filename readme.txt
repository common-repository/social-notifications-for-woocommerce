== Description ==
**IMPORTANT: *Social Notifications for WooCommerce* requires WooCommerce 3.0 or higher.**

**WhatsApp API: Buy WhatsApp credits on [mTalkz](http://www.mtalkz.com)**


**Social Notifications for WooCommerce** sends WhatsApp notifications to your clients for order status changes. You can also receive a WhatsApp message when a new order is received.
 
= Features =  
* Supported WhatsApp gateway: [mTalkz](http://www.mtalkz.com)
* Notify store owner about new orders
* Send WhatsApp notification to client for each order status change
* Customize WhatsApp message template for each order status
* Custom signature can be appended to all outgoing WhatsApp messages
* Support for a large number of standard variables in WhatsApp message templates
* Option to add custom variables in WhatsApp templates
* Send international WhatsApp messages
* Send WhatsApp message to billing or shipping address phone
* Automatically insert the international country code prefix to the customer’s phone number
* Can log all outgoing WhatsApp messages for audit trail

== Installation ==
1. You can:
 * Upload the `wa-notifications-for-woocommerce` folder to `/wp-content/plugins/` directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **Social Notifications for WooCommerce** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Set up plugin on *WooCommerce -> Social Notifications for WooCommerces* or through *Settings* link.
4. For obtaining credentials, please [register here](http://www.mtalkz.com/my-account/)
